开发流程

新建一个页面通常需要创建几个文件：

  • 模板 zapp/SITE/model/模块名.py
  • 视图 zapp/SITE/view/模块名.py
  • Coffee Script 脚本 coffee/SITE/auth/模块名/视图名.coffee
  • Make 模板 html/SITE/auth/模块名/视图名.html
  • CSS 样式文件 css/SITE/auth/模块名.css

并在 _url.py 中注册视图。
