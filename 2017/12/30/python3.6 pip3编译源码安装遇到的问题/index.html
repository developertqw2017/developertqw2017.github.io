今天编译python3.6源码和pip9.0.1源码安装后遇到问题

lsb_release

The issue is that you’re default Python implementation is trying to use Python 3 but lsb_release requires Python 2. To fix this problem do the following:

  • Open /usr/bin/lsb_release (Make sure you use sudo or open as root!)
  • Edit the first line to be #! /usr/bin/python2.7
  • Save the file
